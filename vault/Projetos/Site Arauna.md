# Site Arauna

Landing page institucional da [[../Empresa/Arauna Investimentos]].

## Arquivo
`arauna.html` — página única estática (sem dependências externas)

## Seções
1. **Nav** — logo, links âncora, botão CTA "Falar com especialista"
2. **Hero** — headline, subtítulo, botões primário/outline, stats (R$200M+, 500+ clientes, 15 anos, 98% satisfação)
3. **Sobre** — história, equipe, valores (Transparência, Independência, Ética, Resultados)
4. **Serviços** — 6 cards: Assessoria, Renda Fixa, Renda Variável, Invest. Internacionais, Gestão para Empresas, Aposentadoria
5. **Diferenciais** — 4 métricas: 100% independente, 24h suporte, CVM regulamentado, R$0 primeira consulta
6. **Contato** — info de contato + formulário de consulta gratuita
7. **Footer** — links, aviso legal

## Tecnologia
- HTML + CSS inline (variáveis CSS com paleta dourada `#c9a84c`)
- Sem JavaScript, sem frameworks
- Responsivo (media query `max-width: 768px`)

## Design
- Tema escuro (`#0a0c0f`) com dourado (`#c9a84c`) como cor de destaque
- Fonte: `Segoe UI`, `system-ui`
- Scroll suave entre seções
